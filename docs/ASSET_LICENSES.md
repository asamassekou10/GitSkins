# Asset Licenses

GitSkins uses generated SVG/PNG graphics and open-source avatar systems. This file documents the asset sources that can appear in generated profile pictures and cards.

## DiceBear

- Source: https://www.dicebear.com/
- Packages: `@dicebear/core`, `@dicebear/collection`
- Used for: Real Character avatars in `/avatar` and `/api/avatar?family=dicebear`
- Notes: GitSkins generates avatars locally from the installed npm packages. We do not claim ownership of upstream avatar collections.

## GitSkins Original Renderers

- Source: this repository
- Used for: Abstract, mascot, and developer character avatars generated by `/api/avatar`
- Notes: These are deterministic renderers built from code, theme colors, and username seeds.

## Character Policy

GitSkins should not ship copyrighted characters from shows, games, movies, or brands unless the project has an explicit license to use them. For example, do not add characters such as SpongeBob, Dracula from a specific franchise adaptation, Mario, Marvel/DC characters, or branded mascots as bundled assets.

Use original archetypes, public-domain concepts, or open-source avatar systems with clear licensing instead.
